# C++ Console BattleShipt

2 player battle ship game within console. 

## Compile Project
* go to project directory
* run *make*

## Running Program
* go to project directory
* run *./main*
* follow instructions on console log

## Game Notes

Board Legend
* o   unattacked spot
* x   attacked spot (empty)
* \*   sunken ship
* \-   unattacked ship
* \+   attacked ship
